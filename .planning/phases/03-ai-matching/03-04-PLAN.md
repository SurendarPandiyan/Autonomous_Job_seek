# Plan 03-04: Profile Embedding Celery Task

## Objective

Append `embed_profile(user_id: int)` to `matching/tasks.py`. This task loads the user's
`Profile` and `JobPreferences`, builds the profile text string, calls `get_embedding`, and
stores the resulting vector to `Profile.embedding` and resets `Profile.enriched_at` to `now`.
Then hook `embed_profile.delay(user_id=current_user.id)` into the PATCH profile and PATCH
preferences endpoints so embeddings are regenerated automatically after each user update.
Finally, expose `POST /api/v1/workers/embed-profile` for manual triggering.

## Must Haves

- [ ] `embed_profile` is a Celery task with name `"jobplatform.matching.tasks.embed_profile"`
- [ ] Task accepts `user_id: int` as its single argument
- [ ] After task runs, `Profile.embedding` is a 1536-dim list and `Profile.enriched_at` is a UTC datetime
- [ ] `PATCH /api/v1/users/me/profile` dispatches `embed_profile.delay(user_id=current_user.id)` after committing
- [ ] `PATCH /api/v1/users/me/preferences` dispatches `embed_profile.delay(user_id=current_user.id)` after committing
- [ ] `POST /api/v1/workers/embed-profile` with `{"user_id": N}` returns 202 with `{"task_id": "..."}`
- [ ] `uv run pytest tests/ -q --tb=short` passes

## Tasks

### Task 1: Add embed_profile task to matching/tasks.py

**Files:**
- `src/jobplatform/matching/tasks.py` — append new imports and two new functions

**Instructions:**

Open `src/jobplatform/matching/tasks.py` (created by Plan 03-03). Append the following to the
existing imports section at the top of the file (after the existing imports, before any function
definitions):

```python
from datetime import datetime, timezone

from sqlalchemy import select

from jobplatform.profiles.models import Profile
from jobplatform.preferences.models import JobPreferences
from jobplatform.matching.service import build_profile_text
```

Note: `select`, `AsyncSessionLocal`, `asyncio`, `structlog`, and `get_embedding` are already
imported. Only add what is missing.

After the `embed_jobs_batch` sync task definition, append the async implementation:

```python
async def _embed_profile_async(user_id: int, task_id: str) -> dict:
    async with AsyncSessionLocal() as db:
        profile = await db.scalar(select(Profile).where(Profile.user_id == user_id))
        if not profile:
            logger.warning("embed_profile.no_profile", user_id=user_id)
            return {"user_id": user_id, "status": "no_profile"}

        prefs = await db.scalar(
            select(JobPreferences).where(JobPreferences.user_id == user_id)
        )

        text = build_profile_text(profile, prefs)
        if not text.strip():
            logger.warning("embed_profile.empty_text", user_id=user_id)
            return {"user_id": user_id, "status": "empty_text"}

        embedding = await get_embedding(text)
        profile.embedding = embedding
        profile.enriched_at = datetime.now(timezone.utc)
        await db.commit()

        logger.info("embed_profile.done", user_id=user_id, task_id=task_id)
        return {"user_id": user_id, "status": "ok"}
```

Then append the sync Celery task wrapper:

```python
@celery_app.task(name="jobplatform.matching.tasks.embed_profile", bind=True)
def embed_profile(self, user_id: int) -> dict:
    task_id = self.request.id or "local"
    return asyncio.run(_embed_profile_async(user_id, task_id))
```

**Verify:** `uv run python -c "from jobplatform.matching.tasks import embed_profile; print(embed_profile.name)"` prints `jobplatform.matching.tasks.embed_profile`.

### Task 2: Hook embed_profile into profile/preferences routers and add worker endpoint

**Files:**

- `src/jobplatform/preferences/router.py` — import embed_profile, dispatch after preferences update
- `src/jobplatform/workers/router.py` — add POST /embed-profile endpoint

**Instructions for profiles/router.py:**

Open `src/jobplatform/profiles/router.py`. Add one import after the existing imports:
```python

```

In the `patch_profile` endpoint, after the `return await update_profile(db, current_user.id, data)` line,
the dispatch must happen BEFORE the return. Restructure the endpoint body as:

```python
@router.patch("/profile", response_model=ProfileResponse)
async def patch_profile(
    data: ProfileUpdate,
    current_user: User = Depends(get_current_user),
    db: AsyncSession = Depends(get_db),

    updated = await update_profile(db, current_user.id, data)

    return updated
```
