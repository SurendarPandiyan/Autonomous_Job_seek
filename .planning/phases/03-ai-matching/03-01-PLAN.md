# Plan 03-01: Embedding Service

## Objective

Add `openai` and `tenacity` dependencies, expose `openai_api_key` in settings, and create
`src/jobplatform/matching/service.py` containing every pure function the rest of Phase 3 will
call: `get_embedding`, `build_job_text`, `build_profile_text`, `compute_ats_score`, and
`compute_skill_gaps`. No DB access happens here; these functions are imported by Celery tasks
and the router in later plans.

## Must Haves

- [ ] `from jobplatform.matching.service import get_embedding` succeeds without ImportError
- [ ] `get_embedding` is an `async def` decorated with tenacity retry on `RateLimitError`
- [ ] `build_profile_text` returns a non-empty string given a profile with at least one field set
- [ ] `build_job_text` includes `job.title`, `job.company`, and truncated `job.description`
- [ ] `compute_ats_score` returns `1.0` when all profile skills appear in the job text
- [ ] `compute_skill_gaps` returns only requirement skills absent from profile skills
- [ ] `uv run pytest tests/ -q --tb=short` passes (existing suite unbroken)

### Task 2: Create matching package with service.py

**Files:**
- `src/jobplatform/matching/__init__.py` — empty, marks package
- `src/jobplatform/matching/service.py` — all pure functions for embeddings and text composition

**Instructions:**

Create `src/jobplatform/matching/__init__.py` as an empty file.

Create `src/jobplatform/matching/service.py` with the following exact contents (imports, then
functions in this order):

**Imports section:**
```python
import json

from openai import AsyncOpenAI, RateLimitError
from tenacity import retry, wait_exponential, stop_after_attempt, retry_if_exception_type

from jobplatform.config import settings
from jobplatform.jobs.models import Job
from jobplatform.profiles.models import Profile
from jobplatform.preferences.models import JobPreferences
```

**Function `get_embedding`:**

Async function with signature `async def get_embedding(text: str) -> list[float]`.
Decorate with `@retry(wait=wait_exponential(multiplier=1, min=2, max=60), stop=stop_after_attempt(5), retry=retry_if_exception_type(RateLimitError), reraise=True)`.
Body: instantiate `AsyncOpenAI(api_key=settings.openai_api_key)`, call `await client.embeddings.create(model="text-embedding-3-small", input=text)`, return `response.data[0].embedding` as `list[float]`.

**Function `build_job_text`:**

Sync function with signature `def build_job_text(job: Job) -> str`.
Build a list of parts: start with `f"{job.title} at {job.company} in {job.location}"`.
Append `job.description[:3000]` if `job.description` is truthy.
Append `f"Requirements: {json.dumps(job.requirements)[:1000]}"` if `job.requirements` is truthy.
Return `"\n".join(parts)`.

**Function `build_profile_text`:**

Sync function with signature `def build_profile_text(profile: Profile, prefs: JobPreferences | None) -> str`.
Build a list of parts:
- If `profile.current_role`: append `f"Current role: {profile.current_role}"`
- If `profile.years_experience`: append `f"Years of experience: {profile.years_experience}"`
- If `profile.skills`: append `f"Skills: {', '.join(profile.skills)}"`
- If `profile.experience`: build `roles = [f"{e.get('title', '')} at {e.get('company', '')}" for e in profile.experience if e.get('title')]`, if roles: append `f"Experience: {'; '.join(roles)}"`
- If `prefs` and `prefs.target_roles`: append `f"Target roles: {', '.join(prefs.target_roles)}"`
- If `prefs` and `prefs.technologies`: append `f"Technologies: {', '.join(prefs.technologies)}"`
- If `prefs` and `prefs.locations`: append `f"Preferred locations: {', '.join(prefs.locations)}"`
Return `"\n".join(parts)`.

**Function `compute_ats_score`:**

Sync function with signature `def compute_ats_score(profile_skills: list[str], job_text: str) -> float`.
If `not profile_skills`: return `0.0`.
`job_lower = job_text.lower()`.
`matched = sum(1 for s in profile_skills if s.lower() in job_lower)`.
Return `round(matched / len(profile_skills), 4)`.

**Function `compute_skill_gaps`:**

Sync function with signature `def compute_skill_gaps(profile_skills: list[str], requirements: dict | None) -> list[str]`.
If `not requirements`: return `[]`.
Check `requirements.get("skills")`: if it is a `list`, use it as `req_skills`. Otherwise collect `[v for v in requirements.values() if isinstance(v, str) and len(v) < 60]`.
`profile_lower = {s.lower() for s in profile_skills}`.
Return `[s for s in req_skills if s.lower() not in profile_lower]`.
